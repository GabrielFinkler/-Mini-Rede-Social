# Codigo-base minimo

Esta versao entrega apenas uma estrutura inicial bem leve.

É responsabilidade dos alunos definir:

- as structs principais;
- os campos de cada struct;
- as estruturas auxiliares obrigatorias;
- as funcoes auxiliares;
- a implementacao dos comandos.

Arquivos:

- `minirede.h`: constantes, struct `MiniRede` a completar e prototipos dos comandos.
- `minirede.cpp`: funcoes vazias com `TODO`.


Execucao:

```sh
./minirede < ../testes/publicos/pequeno/entrada.txt
```

Importante: o programa deve ler comandos da entrada padrao e imprimir apenas as
respostas especificadas no enunciado. Nao deve haver menu interativo nem prompt.
