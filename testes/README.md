# Testes do trabalho final

Estrutura:

- `pequeno`: teste publico pequeno.
- `grande`: teste publico maior, cobrindo erros, fila, feed e ranking.

Cada teste possui:

- `entrada.txt`: comandos de entrada.
- `saida_esperada.txt`: saida exata esperada.

Uso recomendado para os alunos:

```sh
python3 testes/corretor.py --alvo .
```
